# Third-Party Notices

No third-party source code or binary artifact has been vendored into this repository yet.

Research-only references are recorded in [`docs/research/public-implementations.md`](docs/research/public-implementations.md). A reference is not a dependency and does not authorize copying its source.

Before third-party code is added, record:

- project and source URL;
- exact version or commit;
- imported files/components;
- SPDX license and required notices;
- local modifications;
- whether the code is linked, vendored, generated, or used only as a test fixture.
